자바스크립트에서 스프레드 문법(spread syntax)은 배열이나 객체를 펼쳐서 개별 요소로 분리하는 문법
주로 배열과 객체의 결합, 복사, 함수 호출 시 인수 전달 등 다양한 상황에서 사용

document.getElementsByTagName('li') HTMLCollection forEach, filter 사용 불가

방법
const liElements = document.getElementsByTagName("li"); // li 태그를 모두 선택
console.log(liElements); // HTMLCollection(3) [li, li, li]

1. for문으로 요소 순회
for (let i = 0; i < liElements.length; i++) {
	console.log(liElements[i].textContent);
}

2. for...of 문 적용
for (const liElement of liElements) {
	console.log(liElement.textContent);
}

3. 배열로 변환
3_1. forEach() 함수 적용
const arr = [...liElements]; // 스프레드 문법으로 배열 개체를 만듦
arr.forEach(li => {
	console.log(li.textContent);
});

3_2 - forEach() 함수 적용
const liElementsArray = Array.from(liElements); // Array.from() 메서드로 배열 개체를 만듦
liElementsArray.forEach(li => {
	console.log(li.textContent);
});

4. Array.prototype에 바인딩하기
Array.prototype.forEach.call(liElements,(element) => {
	console.log(li.textContent);
})

5. HTMLCollection.prototype에 함수 직접 생성
HTMLCollection.prototype.forEach = Array.prototype.forEach;
liElements.forEach((element) =>{
	console.log(li.textContent);
})
